## Installation
<!-- So you can copy and paste it one by one :D -->
```md
[p]load downloader
```
```md
[p]repo add Kuro-Cogs https://github.com/Kuro-Rui/Kuro-Cogs
```
```md
[p]cog install Kuro-Cogs <cog_name>
```
```md
[p]load <installed_cog>
```
***Note**: `[p] is your prefix!`
